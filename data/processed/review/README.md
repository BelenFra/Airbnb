This folder is reserved for cleaned Airbnb review datasets generated by `scripts/cleaning/reviews/run_full_review_cleaning.py`.

Expected output layout (snake_case, per-city subfolders, matching listing/calendar pipelines):
- `data/processed/review/hawaii/reviews_hawaii_cleaned.csv`
- `data/processed/review/los_angeles/reviews_los_angeles_cleaned.csv`
- `data/processed/review/nashville/reviews_nashville_cleaned.csv`
- `data/processed/review/new_york/reviews_new_york_cleaned.csv`
- `data/processed/review/san_francisco/reviews_san_francisco_cleaned.csv`

Plus a merged dataset across cities:
- `data/processed/reviews_all_cleaned.csv`

These processed files are intentionally ignored by Git because they are generated artifacts and too large to store in the repository.

CSV outputs are written with `csv.QUOTE_ALL` (every field quoted, `\\` escape) to keep multi-line review comments from breaking row alignment downstream.
