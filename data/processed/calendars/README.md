This folder is reserved for cleaned Airbnb calendar datasets generated by the calendar cleaning pipeline.

Suggested output naming pattern:
- `hawaii_calendar_cleaned.csv`
- `losangeles_calendar_cleaned.csv`
- `nashville_calendar_cleaned.csv`
- `newyork_calendar_cleaned.csv`
- `sanfrancisco_calendar_cleaned.csv`

These processed files are generated artifacts and may be ignored by Git if they are large.
