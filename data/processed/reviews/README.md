This folder is reserved for cleaned Airbnb review datasets generated by the review cleaning pipeline.

Expected output filenames:
- `hawaii_cleaned.csv`
- `losangeles_cleaned.csv`
- `nashville_cleaned.csv`
- `newyork_cleaned.csv`
- `sanfrancisco_cleaned.csv`

These processed files are intentionally ignored by Git because they are generated artifacts and too large to store in the repository.
